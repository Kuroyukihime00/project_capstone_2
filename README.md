
# 📁 Node.js File Server – Mahasiswa & Dosen Wali

Proyek ini dikembangkan sebagai tugas mandiri berdasarkan tiga video referensi (Node.js Introduction, File Access, dan File URL & Navigation). Sistem menampilkan informasi mahasiswa dan dosen wali berbasis file HTML dan JSON sederhana tanpa koneksi basis data.

## 🎯 Tujuan
- Membuat server Node.js tanpa framework
- Menyajikan file statis HTML/JSON
- Mengatur routing sederhana (tanpa database)
- Menyesuaikan proyek dengan studi kasus mahasiswa & dosen wali

## 📌 Struktur Folder

```
project_capstone_2/
├── node_project/
│   ├── app.cjs                  # Server utama
│   ├── views/
│   │   ├── index.html
│   │   ├── mahasiswa.html
│   │   └── dosen.html
│   ├── data/
│   │   ├── mahasiswa.json
│   │   └── dosen.json
│   ├── public/                  # (optional) CSS/JS/Image
│   ├── package.json
│   └── README.md
```

## 🚀 Cara Menjalankan

1. **Clone atau download** project ini
2. Pastikan sudah terinstall Node.js
3. Jalankan perintah berikut:

```bash
cd node_project
node app.cjs
```
